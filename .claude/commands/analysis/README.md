# Analysis Commands

Commands for analysis operations in Claude Flow.

## Available Commands

- [bottleneck-detect](./bottleneck-detect.md) - Detect and analyze performance bottlenecks
- [COMMAND_COMPLIANCE_REPORT](./COMMAND_COMPLIANCE_REPORT.md) - Generate compliance reports for commands
- [performance-bottlenecks](./performance-bottlenecks.md) - Identify performance bottlenecks
- [performance-report](./performance-report.md) - Generate performance analysis reports
- [token-efficiency](./token-efficiency.md) - Analyze and optimize token usage efficiency
- [token-usage](./token-usage.md) - Monitor token consumption patterns
