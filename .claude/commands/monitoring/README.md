# Monitoring Commands

Commands for monitoring operations in Claude Flow.

## Available Commands

- [agent-metrics](./agent-metrics.md) - Track individual agent performance metrics
- [agents](./agents.md) - Monitor all active agents
- [real-time-view](./real-time-view.md) - Real-time monitoring dashboard
- [status](./status.md) - System status and health checks
- [swarm-monitor](./swarm-monitor.md) - Monitor swarm coordination and health
