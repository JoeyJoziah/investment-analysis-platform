# Optimization Commands

Commands for optimization operations in Claude Flow.

## Available Commands

- [auto-topology](./auto-topology.md) - Automatic topology optimization
- [cache-manage](./cache-manage.md) - Cache management and optimization
- [parallel-execute](./parallel-execute.md) - Parallel task execution
- [parallel-execution](./parallel-execution.md) - Advanced parallel execution strategies
- [topology-optimize](./topology-optimize.md) - Swarm topology optimization
