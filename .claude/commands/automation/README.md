# Automation Commands

Commands for automation operations in Claude Flow.

## Available Commands

- [auto-agent](./auto-agent.md) - Intelligent agent routing and dynamic spawning
- [self-healing](./self-healing.md) - Automatic error detection and recovery workflows
- [session-memory](./session-memory.md) - Cross-session memory persistence and restoration
- [smart-agents](./smart-agents.md) - Smart agent auto-spawning based on file type and complexity
- [smart-spawn](./smart-spawn.md) - Intelligent agent coordination and spawning
- [workflow-select](./workflow-select.md) - Workflow selection and execution
